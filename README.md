Bootstrap + React.js + React-Router 
实现“自适应网页动图演示”中的界面.

其中包括:
- 左边可伸缩的下拉菜单
- 中间操作区的tab页
- 右上方可独立变化的登陆区，
- 此外，网页中的部分元素如table，可以实现按照屏幕大小的适配。

# Installation

```bash
npm i -g yarn
yarn install
```

# Running

### `npm start` or `yarn start`

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000/) to view it in the browser.
